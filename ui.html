삼성페이 사용 방법

1.화면의 "주문하기" 버튼을 눌러주세요.

2.상단의 메뉴 종류 중, 원하는 종류를 찾아 눌러주세요.

3.원하는 음료를 찾았으면, 음료 사진을 눌러주세요.

4.추가하고 싶은 사항이 있으면 누르고 없으면 누르지 않아도 돼요.

(단, "필수"라고 쓰인 사항은 지나치치 말고 선택해요.)

5.하단에 선택한 음료 이름과, 주문 수량이 떠요.만약 같은 음료를 여러 개 주문하고 싶다면, +표시를 눌러주세요.

실수로 너무 많이 주문했으면 -표시를 눌러주세요.

6.맨 아래의 "결제하기"를 눌러주세요.

7.매장에서 식사할지, 포장해 갈지를 선택해 눌러주세요.

8.휴대폰에서 삼성페이를 연 후, 카드를 꽂는 곳에 대 주세요.

9.영수증을 받을지 말지를 선택하기만 하면 끝이에요!
